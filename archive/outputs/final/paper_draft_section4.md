# 논문 4장 초안 (쉬운 버전 — 논문에 바로 넣을 땐 톤 높일 것)

---

## 4.1 데이터셋과 표준화

한국 LEED 인증 건물 **460개**를 분석 대상으로 수집했다. 수집한 스코어카드 PDF는 LEED v2.0, v2.2, v2009, v4, v4.1 5개 버전에 걸쳐 있고, 버전마다 카테고리 구성이 다르다. 모든 버전을 **v5 기준**으로 통일하기 위해 규칙 기반 비율 환산(proportional scaling) 적용.

- 주요 버전: v4(276건), v2009(114건) — 전체 84.8%
- 등급 분포: Gold 235(51.1%), Silver 118(25.7%), Platinum 56(12.2%), Certified 51(11.1%)
- 매핑 규칙 107개 적용 → 크레딧 9,747건 중 79%는 rule, 9%는 카테고리 합계 환산, 12%는 미매핑

---

## 4.2 표준화 후 점수 분포

- v5 환산 총점 평균: **45.3점** (SD 12.6, 범위 22.2~93.2)
- 원본 달성률 vs v5 달성률 차이(drift) 평균: **11.9%**
- drift > 20%인 건물: **47건 (10.2%)** — 주로 v5 신규 인증 또는 비정형 PDF, 표준화 불확실성 ↑

---

## 4.3 예측 모델

- 종속변수: 인증 등급 (Certified/Silver/Gold/Platinum)
- 독립변수: v5 카테고리별 달성률 7개 (ratio_EA/LT/MR/EQ/WE/SS/IP) + 연면적 log + LEED 버전
- 모델: XGBoost (n_est=200, depth=6, lr=0.05)
- 검증: Stratified 5-Fold CV

**결과**:
- CV 정확도: **0.8174 ± 0.0502**
- CV Weighted F1: **0.8157 ± 0.0504**
- Train 지표는 1.0이지만 과적합 지표라 논문에 사용 안 함

## 4.4 SHAP 해석

모델이 내놓은 예측의 근거를 SHAP으로 분해했다. 평균 |SHAP| Top-5:

| 순위 | Feature | 평균 \|SHAP\| |
|------|---------|-------------|
| 1 | ratio_EA (에너지) | 0.8840 |
| 2 | ratio_EQ (실내환경) | 0.6533 |
| 3 | ratio_WE (물) | 0.5895 |
| 4 | ratio_LT (입지/교통) | 0.4377 |
| 5 | LEED Version | 0.2880 |

→ **에너지 카테고리 달성률이 등급을 가장 크게 결정**. 다음이 실내환경, 물, 입지 순서.

## 4.5 Robustness 검증

"subset을 다르게 잘라도 Top feature가 EA인가?" 확인:

| Subset | N | CV 정확도 | Top SHAP |
|--------|---|----------|----------|
| 전체 | 460 | 0.8152 | EA |
| credit_hit > 0.7 | 324 | 0.8333 | EA |
| 신버전 (v4, v4.1) | 324 | 0.8333 | EA |
| 구버전 (v2.x, v2009) | 136 | 0.7865 | EA |

→ 어느 subset이든 **EA가 Top**. 결과 robust.

## 4.6 LangGraph를 활용한 검증 구조

데이터셋 품질을 보장하기 위해 LangGraph 기반 파이프라인 구축:

```
PDF → Rule 매핑 → 수학 검증 → LLM 의미 검증 → 최종 저장
```

- Rule: LEED 공식 루브릭을 수식으로 옮김 (결정론적)
- LLM: "이 매핑 말 되나?" 전문가 관점 검증 (Option A — 점수 바꾸지 않음)
- 최종 점수는 항상 Rule (재현성 보장), LLM 판단은 메타데이터

**75건 LLM 리뷰 표본 결과**:
- Rule 승인 7건 (9.3%) — credit_hit 평균 0.885
- Rule 의문 제기 68건 (90.7%) — credit_hit 평균 0.646

→ credit_hit 높을수록 LLM도 신뢰하는 경향.

## 5.1 한계

1. **LLM 리뷰 75/460 (16.3%)**: 표본 리뷰로 포지셔닝
2. **v2009 크레딧 상세 파싱 불가**: 114건은 카테고리 합계만 사용
3. **LLM threshold 0.8 보수적**: 90.7%가 "거부" 판정, 완화 시 결과 달라질 수 있음
4. **구버전 샘플 수**: v2.0 4건, v2.2 18건 — 버전별 SHAP 신뢰도 제한
