# 03. 분석 방법론 (XGBoost + SHAP)

## 연구 질문

**한국 LEED 건물에서 어떤 카테고리가 등급을 결정하는가?**

## 모델

- **모델**: XGBoost Classifier
- **하이퍼파라미터**: n_estimators=200, max_depth=6, learning_rate=0.05
- **Feature**: 9개
  - 카테고리 달성률 7개: `ratio_LT, ratio_SS, ratio_WE, ratio_EA, ratio_MR, ratio_EQ, ratio_IP`
  - `log_area` (연면적 log)
  - `version_ord` (버전 순서 인코딩: v2.0=0, v2.2=1, v2009=2, v4=3, v4.1=4)
- **Target**: `certification_level` (Certified/Silver/Gold/Platinum)

## 검증

- **5-Fold Stratified CV** (등급 분포 보존)
- NaN은 `fillna(0)` (v2009/v2.2의 ratio_SS 141건)

## 성능

| 지표 | 값 |
|------|-----|
| **CV Accuracy** | **0.8174 ± 0.0502** |
| **CV Weighted F1** | **0.8157 ± 0.0504** |
| Train Accuracy | 1.0 (과적합 — 논문 미사용) |

> **Train 지표는 과적합이므로 논문에는 CV 값만 사용**.

## SHAP 분석

- **방법**: TreeExplainer (XGBoost 전용)
- **해석**: 각 feature가 예측에 미치는 평균 기여도 (mean |SHAP|)

### Top 5 결과

| 순위 | Feature | 평균 \|SHAP\| |
|------|---------|-------------|
| 1 | **ratio_EA (에너지)** | **0.8840** |
| 2 | ratio_EQ (실내환경) | 0.6533 |
| 3 | ratio_WE (물) | 0.5895 |
| 4 | ratio_LT (입지/교통) | 0.4377 |
| 5 | version_ord | 0.2880 |

### 해석

**"에너지(EA)가 한국 LEED 등급을 가장 크게 결정"**

- v5에서 EA 만점 33점 (가장 큰 비중)
- 한국 건물들이 에너지 절약에 점수 집중
- 설계 단계 EA 확보가 등급 업그레이드 효과적

## Robustness 검증

subset을 다르게 잘라도 Top feature가 EA인지 확인:

| Subset | N | CV Acc | Top SHAP |
|--------|---|--------|----------|
| 전체 | 460 | 0.8152 | ratio_EA |
| credit_hit > 0.7 | 324 | 0.8333 | ratio_EA |
| 구버전 (v2.x, v2009) | 136 | 0.7865 | ratio_EA |
| 신버전 (v4, v4.1) | 324 | 0.8333 | ratio_EA |
| LLM 리뷰 75건 | 75 | 0.7067 | ratio_EA |

→ **어느 subset에서든 EA가 Top**. 결과 robust ✅

## 한계

1. **Train vs CV gap 18%** → 과적합. Holdout test set 없음
2. **Feature 9개로 제한** → 건물 용도/위치/연도 등 반영 안 됨
3. **CV fold 간 분산 큼** (std 0.0502, fold별 76~89%)
4. **v2009 크레딧 파싱 불가** (114건) → 카테고리 레벨만
